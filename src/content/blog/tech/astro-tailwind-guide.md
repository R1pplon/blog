从零开始搭建一个基于 Astro 和 Tailwind CSS v4 的个人博客。

## 项目初始化

使用 `create-astro` 初始化项目，选择空白模板和 TypeScript。

## 配置 Tailwind v4

Tailwind v4 使用 `@theme` 语义化 token 注册颜色变量，配合 CSS 自定义属性实现昼夜切换。

## 内容管理

使用 Astro Content Collections 管理博客文章，支持 Markdown 和 MDX。
